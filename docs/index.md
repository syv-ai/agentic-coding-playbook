# Agentic Coding Playbook

A hands-on workshop playbook for working effectively with AI coding agents — Claude Code, Cursor, Codex, and others. It is tool-agnostic: the focus is on transferable skills, not a single vendor.

## Two tracks

- **Developer track** — the practical craft: setting up an environment, context engineering, prompting, verification, subagents, and production workflows.
- **Leadership track** — adoption, work allocation, observability, and the organizational side of agentic coding.

Both share a common foundation in the early modules.
